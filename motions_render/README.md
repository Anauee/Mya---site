# Mya Motion - Disparo Inteligente

A HyperFrames video composition for the Mya CRM Workflow section.

## Requirements

- **Node.js 22+**
- **FFmpeg**

## Preview

```bash
npx hyperframes preview
```

## Render

```bash
npx hyperframes render index.html -o disparo_inteligente.mp4 --resolution 1080x810
```
